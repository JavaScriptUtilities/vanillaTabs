# Vanilla Tabs

Basic JS Tabs

## How to install

First, load js/vanilla-tabs.js in your webpage.

Then, when DOM is ready, start the plugin :

```
<script>
new vanillaTabs({
    triggers: document.querySelectorAll('.dk-jsu-triggers a'),
    targets:  document.querySelectorAll('.dk-jsu-targets p')
});
</script>
```
